# IT Support and Technical Procedures

## Network and Connectivity
### VPN Access
- Company VPN required for remote access
- Download VPN client from IT portal
- Two-factor authentication required
- Connection issues: restart VPN client, then reboot
- Supported devices: Windows 10+, macOS 10.14+, iOS 12+, Android 8+

### Wi-Fi Access
- Office networks: Corp-Secure (employees), Corp-Guest (visitors)
- Corp-Secure requires employee credentials
- Wi-Fi password available from IT helpdesk
- Connection issues: forget network, reconnect, contact IT if persistent

### Password Management
- Password complexity: minimum 12 characters, include numbers and symbols
- Password expiration: every 90 days
- Use password manager (LastPass enterprise license provided)
- Account lockout after 5 failed attempts (30-minute reset)
- Self-service password reset available

## Hardware Support
### Device Provisioning
- New employees receive laptop within first week
- Standard laptop: Dell Latitude or MacBook Pro based on role
- Additional monitors available upon request
- Docking stations provided for office workstations

### Common Hardware Issues
**Laptop won't turn on:**
- Check power adapter connection
- Try different power outlet
- Remove and reinsert battery (if removable)
- Contact IT if issue persists

**Slow performance:**
- Restart computer (first troubleshooting step)
- Clear cache and temporary files
- Check for malware using company antivirus
- Consider hardware upgrade if 3+ years old

**Display issues:**
- Update graphics drivers
- Check display connections
- Try external monitor
- Calibrate display settings

## Security Procedures
### Access Control
- Multi-factor authentication required for all systems
- Role-based access control implemented
- Regular access reviews conducted quarterly
- Immediate access revocation upon termination

### Data Protection
- Company data encryption mandatory
- Cloud storage approved: OneDrive, Google Drive
- USB drives: encrypted only, approved by manager
- Data classification: Public, Internal, Confidential, Restricted

### Incident Reporting
- Security incidents: report immediately to IT Security
- Suspicious emails: forward to phishing@company.com
- Lost/stolen devices: report within 1 hour
- Security hotline available 24/7

## Help Desk Procedures
### Ticket Submission
- Submit tickets through IT helpdesk portal
- Priority levels: Critical (4-hour response), High (8-hour), Normal (24-hour)
- Include detailed description and error screenshots
- Request confirmation ticket number

### Common Resolution Times
- Password reset: 15 minutes
- Software installation: 2-4 hours
- Hardware replacement: 1-2 business days
- Network connectivity: 1-4 hours depending on complexity
